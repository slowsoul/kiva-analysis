Problem statement Kiva loans

Kiva is a non-profit organization with a vision to connect people through lending to alleviate poverty. Leveraging the internet and a worldwide network of microfinance institutions, Kiva lets individuals help create opportunity around the world.

Basically, a user of Kiva finds a loan, lends as little as $25 to help fund the loan which is vetted, administered and finally disbursed to borrowers by field partners of Kiva. Lenders will get updates about loans they have helped and get money back as the borrower repays the loan.

Data source and possible analysis

Kiva created a set of API so that anyone can help expand the reach and create new features for the Kiva community. Data snapshots are also provided for uses such as analysis and visualization. The latest snapshot was generated on Feb 13, 2014, the download link is http://s3.kiva.org/snapshots/kiva_ds_json.zip. As for the loan part, there are 667467 records stored in 1335 json files. One standard json file have 500 records inside. The explanation of fields in a loan record can be found here

Each record includes:
  • Borrower(s) information
  • Sector and description of planned usage
  • Loan amount and funded amount
  • Number of lenders
  • Repayment terms and schedule
  • Repayment records
  • Journals
With this large amount of data, we can find out:
  • Total loan amount and average loan amount for each country
  • Loan sector distribution for all loans and for loans in one specified country
  • Average number of lenders contributing to a loan in each year
  • Average number of borrowers in a loan in each year
  • The status distribution for loans posted in each year
  • etc.
